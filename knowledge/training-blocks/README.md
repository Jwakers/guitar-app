# Training Blocks

This directory contains one document per training block type.

Training blocks are multi-week structured programmes. Each block type represents a distinct training philosophy and weekly structure. The adaptive engine uses block documents as the basis for generating weekly plans and session prescriptions.

## Document Template

Each training block document must define the following sections:

```
# [Block Name]

## Block Objective
The primary training goal of this block type.

## Typical Duration
Recommended number of weeks.

## Weekly Structure
The pattern of session types across a typical week
(e.g. standard / standard / light / standard / test).

## Target Workload
Low | Moderate | High — overall training load for this block.

## Primary Skills
The skill areas this block develops most intensively.

## Secondary Skills
Supporting skills trained at lower volume.

## Progression Philosophy
How the block increases difficulty across its duration.

## Deload Strategy
When and how a deload week is introduced within this block.

## Completion Criteria
The conditions that indicate a player has completed this block successfully.

## Typical Next Block
The recommended block type to follow this one.
```
